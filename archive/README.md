# Archive

Archived code and files that are no longer in active use but preserved for reference.

## Contents

### `/analysis/`
Original correlation analysis scripts from initial MVP (Dec 17, 2024).
- `correlator.py` - Tweet-price correlation calculations
- `visualize.py` - Plotly visualization generator

These were replaced by the interactive web chart component.

### `/output/`
HTML visualization outputs from the analysis scripts (~23MB).
- `correlation_scatter.html`
- `dashboard.html`
- `price_vs_tweets.html`
- `quiet_periods.html`
- `tweet_day_comparison.html`

### `/web/src/lib/useMobile.ts`
React hook for detecting mobile viewport. Never imported/used.
Archived in case mobile-specific behavior is needed in the future.
